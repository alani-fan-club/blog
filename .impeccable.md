# Design Context — Alani Fan Club Research Blog

## Target Audience
AI/ML security researchers, collaborators, and curious engineers following our work. Also consumed by AI agents (clawdbots) via RSS feeds.

## Use Cases
- Read research updates, roundups, and informal posts from the group
- Browse active research projects (SpecSec, Assistant-LoRA)
- Subscribe via RSS/Atom feed (both human readers and AI agents)

## Brand Personality / Tone
**Dark, minimal, commanding.** Inspired by the Marathon (Bungie) UI design language — specifically the work by Michael Rigley / Antibody studio:
- Deep black backgrounds (#000 or near-black)
- Bold, geometric or monospace typography — think sci-fi terminal interfaces
- Minimal color palette: one or two accent colors max (cyan/turquoise #3bceac or electric blue)
- Clean grid with generous negative space
- Subtle scan-line or CRT texture effects (CSS only, not distracting)
- Interface-like section dividers — thin rules, dashes, or terminal-style separators
- No decorative images — pure typography and layout
- The vibe is "research lab terminal" not "tech startup blog"

## Differentiation
This should feel like reading dispatches from a research bunker. Not a Medium clone. Not a corporate blog. It should feel like the interface of a ship's AI broadcasting findings into the void. The Marathon aesthetic — bold color blocks against black, functional typography, glitch-adjacent texture — but restrained and readable.

## Technical Constraints
- Static site generator (Astro preferred — modern, fast, markdown-native)
- All content as Markdown files in `/posts/` and `/projects/`
- RSS/Atom feed auto-generated
- GitHub Pages compatible (static HTML output)
- No JavaScript frameworks for the reader — pure HTML/CSS with minimal JS for effects
- Must look good on mobile
- CNAME file for custom domain support
- GitHub Actions workflow for auto-deploy on push to main
